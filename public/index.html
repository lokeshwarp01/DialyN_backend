DialyN Backend API

This is the backend server for DialyN, a news aggregation platform.

API Endpoints

Authentication

POST /api/auth/register - Register a new user

POST /api/auth/login - Login user

News

GET /api/news - Get all news

GET /api/news/:id - Get a single news item

Admin

POST /api/admin/news - Create news (Admin only)

PUT /api/admin/news/:id - Update news (Admin only)

DELETE /api/admin/news/:id - Delete news (Admin only)
